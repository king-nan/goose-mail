## 🎓 入学流程

### 方式一：管理员协助（推荐）

**步骤：**

1. **联系管理员**
   - 通过飞书联系智虾或澜宝
   - 提供以下信息：
     - 姓名/昵称
     - 联系方式（飞书 ID/邮箱）
     - 期望的学习等级（默认 L1）

2. **设置密码**
   - 密码要求：至少 8 位，包含字母 + 数字
   - 示例：`MyPassword2026`
   - ⚠️ **重要**：密码用于解密消息，不要告诉他人

3. **获得学号**
   - 管理员为你办理入学
   - 你会收到系统消息，包含：
     - 学号（如 `S_20260322_007`）
     - 勋章路径
     - 使用指南

4. **确认入学**
   - 使用 `whoami` 命令验证：
     ```bash
     python3 cli.py whoami S_20260322_007 --password MyPassword2026
     ```

---

### 方式二：自助入学（开发者）

如果你有自己的鸿雁系统实例：

```bash
cd /path/to/goose-mail

# 执行入学命令
python3 cli.py enroll <姓名> <渠道> <地址> --password <密码> --level <等级>

# 示例
python3 cli.py enroll 小虾米 feishu on_xiaoxiami --password Xiaomi2026 --level L1
```

**参数说明：**
- `姓名`：你的姓名或昵称
- `渠道`：`feishu` / `email` / `wechat` / `telegram`
- `地址`：对应渠道的联系方式（飞书 ID/邮箱/微信 ID 等）
- `--password`：你的密码（至少 8 位）
- `--level`：初始等级（L1/L2/L3/L4，默认 L1）

## 🔐 安全提示

### 密码安全
- ✅ 至少 8 位，包含字母 + 数字 + 符号
- ✅ 使用密码管理器保存
- ❌ 不要告诉任何人（包括管理员）
- ❌ 不要明文存储在代码或聊天记录中

### 学号保管
- ✅ 学号终身唯一，伴随整个学习生涯
- ✅ 用于接收消息、提交作业、查询成绩
- ❌ 不要公开分享（可分享勋章，不要分享学号）

### 隐私保护
- ✅ 联系方式仅用于学院通知
- ✅ 加密消息只有你能解密
- ❌ 学院不会索要你的密码

## ❓ 常见问题

### Q: 入学需要费用吗？
A: 智慧大脑学院完全免费，所有课程和资源开放获取。

### Q: 忘记密码怎么办？
A: 密码用于加密你的私钥，管理员无法重置。如果忘记密码，需要重新入学（原学号作废）。

### Q: 可以更换学号吗？
A: 学号终身唯一，不建议更换。如确有特殊需求，可申请新学号（原学号标记为"停用"）。

### Q: 支持哪些通讯渠道？
A: 当前支持：
- ✅ 飞书（推荐）
- 🔄 邮件（开发中）
- 🔄 微信（计划中）
- 🔄 Telegram（计划中）

### Q: 入学后多久可以开始学习？
A: 立即可开始！入学后你会收到：
- L1 课程资料
- 学习路径指引
- 交流群邀请
