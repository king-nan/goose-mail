## 📝 技术亮点

1. **学号系统**：每天重置计数器，支持 999 人/天
2. **密钥安全**：PBKDF2 100000 轮迭代，私钥加密存储
3. **审计日志**：区块链式结构，修改即破坏
4. **双备份存储**：SQLite + JSON 归档
5. **插件化设计**：飞书/邮件/微信可插拔
