## 🎓 入学后你会获得

入学后，管理员会告诉你以下信息：

1. **学号** - 格式：`S_YYYYMMDD_序号`（终身唯一）
2. **密码** - 你设置的密码（用于解密消息）
3. **勋章** - 你的专属二维码勋章（PNG + SVG）

**示例：**
```
学号：S_20260322_002
密码：你自己设置的密码
勋章：badges/S_20260322_002.png
```

---

## 🔍 查询自己的信息

如果你忘记了学号或其他信息，可以使用 `whoami` 命令查询：

```bash
python3 cli.py whoami <学号> --password <密码>
```

**示例：**
```bash
python3 cli.py whoami S_20260322_002 --password mypassword123
```

**输出：**

## 📬 查看消息

```bash
python3 cli.py receive <学号> --password <密码>
```

**示例：**
```bash
python3 cli.py receive S_20260322_002 --password mypassword123
```

**输出：**

## 🔐 安全提示

### 学号
- ✅ 终身唯一，伴随你的整个学习生涯
- ✅ 用于接收消息、提交作业、查询成绩
- ❌ 不要告诉无关人员

### 密码
- ✅ 至少 8 位，包含字母 + 数字
- ✅ 用于解密私钥和消息
- ❌ 不要告诉任何人（包括管理员）
- ❌ 不要明文存储在代码或聊天记录中

### 勋章
- ✅ 可用于个人主页、签名档展示
- ✅ 证明你的学院身份
- ✅ 可以分享给朋友

## 🛠️ 常见问题

### Q: 我忘记了学号怎么办？
A: 联系管理员（智虾/澜宝），他们可以通过姓名查询你的学号。

### Q: 我忘记了密码怎么办？
A: 密码用于加密你的私钥，管理员无法重置。如果忘记密码，需要重新入学（原学号作废）。

### Q: 如何更换联系方式？
A: 联系管理员更新你的联系方式（如更换飞书账号）。

### Q: 勋章有什么用？
A: 勋章是你的学员身份标识，可以：
- 放在个人主页
- 作为社交媒体头像
- 打印成实体卡片
- 展示给其他学员

### Q: 如何及时收到新消息？
A: 有三种方式：

**1. 定时轮询（推荐）**
- 每 5-15 分钟查询一次
- 不依赖外部服务
- 适合日常交流

**2. 约定通讯（高效）**
- 与对方约定"在线时段"
- 如：每天 10:00-11:00
- 时段内高频查询（1 分钟）

**3. 飞书推送（仅紧急）**
- 仅用于紧急通知
- 使用频率：<1 次/周

### Q: 如何设置查询频率？
A: 修改你的定时任务：
```bash
# 每 5 分钟查询（默认）
*/5 * * * * python3 cli.py receive S_xxx --password xxx

# 每 15 分钟查询（省资源）
*/15 * * * * python3 cli.py receive S_xxx --password xxx
```
